# ProjectForge

Production-oriented Next.js planning workspace for engineers to design complete software projects before implementation.

## Stack
Next.js 15, TypeScript, TailwindCSS, shadcn-style primitives, Framer Motion, React Hook Form, Zod, TanStack Query, Supabase-ready auth/data layer, Prisma/PostgreSQL, Recharts, and Lucide.

## Run
```bash
npm install
npm run dev
```
